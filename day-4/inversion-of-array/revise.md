# Inversion of Array

## Core Trigger

Count pairs:

arr[i] > arr[j]

and

i < j

---

## Optimal Approach

Merge Sort

Count inversions during merge step.

---

## Pattern

Merge Sort

Divide and Conquer

---

## Complexity

TC: O(N log N)

SC: O(N)

---

## Memory Trick

Divide

↓

Sort Left

↓

Sort Right

↓

Merge

↓

Count Inversions
