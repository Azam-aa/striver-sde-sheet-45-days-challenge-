# Linked List Cycle II

## Core Trigger

Find starting node of cycle.

---

## Approach

Detect cycle first.

Move slow to head.

Move both one step.

Meeting point = cycle start.

---

## Pattern

Floyd Cycle Detection

---

## Complexity

TC: O(N)

SC: O(1)

---

## Memory Trick

Detect Cycle
↓
Reset Slow
↓
Move Together
↓
Meet at Start
