# Flattening Linked List

## Core Trigger

Multiple sorted linked lists.

Flatten into one list.

---

## Approach

Flatten right side.

Merge recursively.

---

## Pattern

Merge Lists

Recursion

---

## Complexity

TC: O(N*M)

SC: O(Recursive Stack)

---

## Memory Trick

Flatten Right
↓
Merge Left
↓
Return Flat List
