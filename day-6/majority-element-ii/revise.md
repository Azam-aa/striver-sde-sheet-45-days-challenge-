# Majority Element

## Core Trigger

Element appears more than N/2 times.

---

## Approach

Moore's Voting Algorithm.

Maintain candidate and count.

---

## Pattern

Voting Algorithm

---

## Complexity

TC: O(N)

SC: O(1)

---

## Memory Trick

Count = 0

↓

Pick Candidate

↓

Same → Count++

Different → Count--
