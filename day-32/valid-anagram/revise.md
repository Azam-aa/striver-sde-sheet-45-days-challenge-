# Valid Anagram

Approach:
Frequency Array

Pattern:
Hashing

TC:
O(N)

SC:
O(1)

Points:
- Lengths must be equal.
- Increase count for first string.
- Decrease count for second string.
- Every frequency should become zero.
- Otherwise return false.

Edge Cases:
- Different lengths.
- Empty strings.
- Same strings.
- Repeated characters.
